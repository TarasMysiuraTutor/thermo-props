# H₂O Thermo Calc

Веб-калькулятор теплофізичних властивостей води на основі апроксимаційних формул IAPWS-IF97. Підтримує лід Ih, рідку воду та перегріту пару (діапазон −100…800 °C, тиск 0.001–100 МПа).

## Стек

- **React 18** + **Vite 5** — SPA без бекенду, усі обчислення в браузері
- Звичайний CSS (`src/styles.css`) із підтримкою світлої / темної / авто тем
- Власна локалізація через React Context: **UA / EN / PL / DE**
- Збереження історії розрахунків, теми та мови у `localStorage`

## Структура проєкту

```
index.html              точка входу Vite (підключає /src/main.jsx)
vite.config.js          конфіг Vite (порт 5000, host 0.0.0.0)
package.json            залежності та скрипти
public/                 статичні ресурси (favicon, QR-зображення)
src/
  main.jsx              корінь React + I18nProvider
  App.jsx               головний стан (поля, історія, результат)
  styles.css            оригінальні стилі
  water.js              формули IAPWS-IF97 (ES-модуль)
  i18n.jsx              переклади + React Context (`useI18n`)
  version.js            константа APP_VERSION
  components/
    Header.jsx          логотип, перемикач теми, перемикач мови
    Hero.jsx            заголовок
    InputCard.jsx       форма температури/тиску, одиниці, помилки
    HistoryPanel.jsx    останні 10 розрахунків
    Results.jsx         фаза, T_насич / T_замерз, сітка властивостей
    Chart.jsx           SVG-графік, підказка при наведенні, експорт PNG
    ExportBar.jsx       експорт CSV і PDF (через друк)
    Footer.jsx          посилання на IAPWS + версія
```

## Запуск локально

Потрібен **Node.js 18+**.

```
npm install        # встановити залежності (один раз)
npm run dev        # сервер розробки на http://localhost:5000
npm run build      # продакшн-збірка → dist/
npm run preview    # локальний перегляд готової збірки
```

## Розгортання на Replit

- Воркфлоу `Start application` запускає `npm run dev` на порту 5000.
- Публікація налаштована як **статичний сайт**: `npm run build` → тека `dist`.
- Натисніть **Publish** угорі робочого простору, щоб отримати безкоштовну адресу `*.replit.app`.

### v1.9.0
- Додано **підтримку фази льоду** (Ih). При T < 0.01 °C автоматично відображається набір властивостей льоду.
- Властивості льоду:
  - **ρ** — густина (Pounder): 916.7 − 0.1403·T_C
  - **cₚ** — теплоємність (підгонка до IAPWS R10-06): 2096.6 + 6.49·T_C
  - **λ** — теплопровідність (Andersson & Inaba): 632/T_K + 0.38 − 0.00197·T_K
  - **a** — температуропровідність (λ/(ρ·cp))
  - **β** — об'ємний коефіцієнт теплового розширення
  - **c** — швидкість звуку (поздовжня)
  - **h** — питома ентальпія (відносно рідкої води при 0 °C)
  - **L_sf** — питома теплота плавлення (333.55 кДж/кг)
- Розширено допустимий діапазон температури з [0; 800] до **[−100; 800] °C** (відповідно K і °F).
- Графік тепер відображає три гілки: **лід (блакитна)** — рідина (синя) — пара (червона), без штучного з'єднання при стрибках.
- Підказка на графіку та позначка фази підсвічуються відповідним кольором.
- Додано нові i18n-ключі (prop.hsf, unit.hsf, chart.legendIce) для 4 мов.

### v1.6.0
- Набір властивостей тепер залежить від фази:
  - **Рідка вода**: ρ, cp, λ, a, μ, ν, Pr, σ, β, Pₛ, h, **r** (теплота пароутворення при T_насич для поточного P).
  - **Перегріта пара**: ρ, **v** (питомий об'єм), cp, **γ** (показник адіабати), λ, a, μ, ν, Pr, β, **c** (швидкість звуку), h, **r** (теплота пароутворення).
- Для пари прибрано поверхневий натяг та тиск насиченої пари — у газовій фазі вони не мають фізичного сенсу.
- Теплота пароутворення обчислюється за кореляцією Watson (h_fg = 262.1·(Tc − T)^0.38), коректна аж до критичної точки 374 °C.
- Швидкість звуку та γ для пари — за моделлю ідеального газу (γ = cp/cv, c = √(γ·R_s·T)).
- Додано переклади нових властивостей для всіх чотирьох мов.
